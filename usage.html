modelos.read_boxes_in_easy_ocr_and_transformer(asset, cajas, lang='es')#

Applies OCR techniques to the assset argument using cajas as delimiter.

Parameters

  • asset (str) – Absolute FilePath of document to analize. Must be: .jpg .jpeg or .png

  • cajas (list) – A list containing four point regions where text is to be recognized

  • lang (str, optional) – pecifies language to recognize. More options can be found here: https://www.jaided.ai/easyocr/, defaults to “es”

Returns

A string with text detected in asset

Return type

str
